# Презентации МК №1 и МК №2

Единый HTML-файл для показа на экране во время мастер-классов.

**URL на GitHub Pages:** https://chetverda.github.io/AI_for_teachers/presentations/

## Управление

| Действие | Клавиша |
|---|---|
| Следующий слайд | → или пробел |
| Предыдущий слайд | ← |
| На главную (выбор МК) | Esc или кнопка «Главная» |
| Полный экран | F11 |

## Содержание

- **МК №1** — 19 слайдов (6 миссий, 12:15–14:00)
- **МК №2** — 16 слайдов (5 фаз, 15:00–16:30)

## Загрузка на GitHub

Загрузите папку `presentations/` в репозиторий (рядом с `index.html` портала):

```
AI_for_teachers/
  index.html          ← портал материалов
  presentations/
    index.html        ← презентации
    assets/
      deck.css
      deck.js
      prompts.js
      fonts/
```

По одному файлу, если браузер снова выдаёт ошибку — см. `КАК_ЗАГРУЗИТЬ.md` в корне.

## Пересборка

После правок в `МК-01/03_Презентация_участникам.html` или `МК-02/...`:

```bash
python3 "СтажИИровка для учителей 2026/портал/scripts/build_presentations.py"
```

Файлы `deck.css` и `deck.js` правятся в `presentations/assets/` — скрипт их не перезаписывает.
